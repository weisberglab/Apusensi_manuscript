# Apusensi_manuscript
This repository contains codes associated with the following manuscript:

Rahman A, Schiffer AM, Fuqua C, Woods RJ, Weisberg AJ. **Genomic insights into niche adaptation and the opportunistic infection potential of Agrobacterium pusense**

- `R` contains R script for analyzing pan genome.
- `data` contains orthologous gene cluster presence-absence data.
